# Time-Table

This repository includes a **browser-only timetable generator** suitable for [GitHub Pages](https://pages.github.com/).

## GitHub Pages app

See [apps/timetable-pages/README.md](apps/timetable-pages/README.md) for the static SPA (`npm run dev`, tests, and deployment).

The original Django/school timetable slice lives under `backend/school/` and `frontend/`.
